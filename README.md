## 💡 Hintergrund & Pädagogisches Konzept
**Mathe-Hüpfer** ist nicht nur ein Spiel, sondern ein gezieltes Belohnungssystem für den Matheunterricht. Die Grundidee ist simpel: Kinder lieben Jump'n'Run-Spiele (wie *Doodle Jump*). Wenn man diese Spielmechanik mit Kopfrechnen verknüpft, üben die Kinder völlig freiwillig und hochmotiviert.

### Warum dieses Spiel funktioniert:
- **Visuelles Lernen:** Aufgabe und Ergebnis stehen gut lesbar als Einheit auf den Plattformen.
- **Logische Konsequenz:** Wer richtig rechnet, springt hoch. Wer sich verrechnet, bei dem bricht die Plattform.
- **Impulskontrolle:** Verlockende Power-Ups (Gadgets) tauchen auch auf falschen Plattformen auf. Das Kind muss lernen, kurz innezuhalten und zu rechnen, statt blind auf die Belohnung zu springen.
- **Drei Schwierigkeitsgrade:** Perfekt anpassbar an den aktuellen Schulstoff (Plus/Minus, Einmaleins, Gemischt).

---

## 🎮 So wird gespielt
Das Ziel ist es, den Ball so hoch wie möglich springen zu lassen. Der Ball springt automatisch – du steuerst nur nach **links** und **rechts**.

1. Schau dir die **Matheaufgabe** auf den Plattformen an.
2. Lenke den Ball auf die Plattform mit dem **richtigen Ergebnis**.
3. **Sammle Gadgets** auf richtigen Plattformen ein, um Boni zu erhalten.
4. Vermeide die falschen Plattformen, sonst stürzt du ab!

### 🎁 Die Gadgets (Power-Ups)
* 🚀 **Rakete:** Trägt dich mit Höchstgeschwindigkeit mehrere Etagen nach oben.
* 🌀 **Superfeder:** Katapultiert dich mit einem riesigen Sprung in die Höhe.
* 🛡️ **Schutzschild:** Gibt dir eine zweite Chance! Landest du auf einer falschen Plattform, bricht sie zwar, aber du darfst trotzdem abspringen.
* ⭐ **Stern:** Verdoppelt deine gesammelten Höhenpunkte für kurze Zeit.
* ❄️ **Zeitlupe:** Das Spiel wird langsamer. Du hast mehr Zeit, die Aufgaben zu lesen und zu rechnen.
* ☁️ **Wolke:** Eine sichere Plattform zum Durchatmen – ganz ohne Rechenaufgabe.

## 📱 Installation als App (PWA)
Der Mathe-Hüpfer ist eine **Progressive Web App (PWA)** und kann ohne App-Store direkt auf Smartphones und Tablets installiert werden. Er funktioniert danach sogar **offline**!

**Auf dem iPhone / iPad (Safari):**
1. Öffne den Link zum Spiel im Safari-Browser.
2. Tippe unten in der Mitte auf das **Teilen-Symbol** (Viereck mit Pfeil nach oben).
3. Wähle **"Zum Home-Bildschirm"**.
4. Tippe auf "Hinzufügen". Die App ist nun auf deinem Startbildschirm.

**Auf Android (Chrome):**
1. Öffne den Link im Chrome-Browser.
2. Tippe oben rechts auf die **drei Punkte** (Menü).
3. Wähle **"Zum Startbildschirm hinzufügen"** oder "App installieren".

## 🛠️ Technische Details
- **Technologie:** 100% Vanilla HTML5, CSS3 und JavaScript (Canvas API).
- **Audio:** Töne werden dynamisch über den `AudioContext` (Synthesizer) generiert. Es werden keine externen mp3-Dateien benötigt, was extrem schnelle Ladezeiten garantiert.
- **Datenschutz:** Es werden keine Benutzerdaten gesammelt. Highscores werden rein lokal im Browser des Geräts (`localStorage`) gespeichert.
